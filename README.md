# Cake Boutique Web Application

A full-stack web application for managing a cake shop, developed using Flask and MySQL.

## 📌 Features
- User authentication (login/signup for customers and employees)
- Role-based access (Admin/Manager & Customer)
- Cake management (add, edit, delete cakes)
- Image upload and management
- Inventory and ingredient tracking
- Order management system
- Shopping cart functionality
- Employee management system

## 🛠 Technologies
- Python (Flask)
- MySQL
- HTML, CSS
- Flask-MySQLdb

## 📂 System Modules
- Authentication system
- Cake & inventory management
- Order processing system
- Employee management
- Shopping cart
